Sources of Taxonomic Data and Software

Taxonomic names and synonymy

The iPlant TNRS currently accesses the following sources for information on plant names and synonymy:
  • Tropicos — Missouri Botanical Garden's online database, providing information on plant names, literature and specimens, with emphasis on the Americas.
    Data currently used by the TNRS are a partial snapshot accessed via a custom development download on 18 August 2010. In future, names will be accessed and updated frequently via the Tropicos API.

Additional sources of names, synonymy and classification will be added in the near future. See Future Developments.

Software

Code from the following open source applications was used during the development of the TNRS:
  • Original Taxamatch Algorithm — Developed by Tony Rees (CSIRO).
  • Taxamatch Web Service (a PHP Port) — Special thanks for assistance from Michael Giddens.
  • Global Names Index (Parser) — Special thanks for assistance from Dmitry Mozzherin.
  • Algorithm for accepted name — Developed by the Missouri Botanical Garden for their Tropicos database.
